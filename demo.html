🚀 Quick Installation

  1. Open Chrome Extensions
    Navigate to chrome://extensions/ in your browser
  2. Enable Developer Mode
    Toggle the "Developer mode" switch in the top right corner
  3. Load Extension
    Click "Load unpacked" and select the extension folder
  4. Go to Google Takeout
    Visit https://takeout.google.com/manage/archive
  5. Start Downloading!
    Click the extension icon and press "Download All"

⚙️ Key Features Explained

🔍 Auto-Detection

The extension automatically scans the page for links containing "download" or "descargar" text:

const linkText = link.textContent.trim().toLowerCase(); if (linkText.includes('descargar') || linkText.includes('download')) { // Found a download link! }

🎯 Visual Highlighting

Download links are highlighted with green borders and background:

element.style.border = '2px solid #4CAF50'; element.style.backgroundColor = '#e8f5e9';

⬇️ Staggered Downloads

Downloads are triggered with a 1-second delay to prevent browser overload:

setTimeout(() => { triggerDownload(url); }, index * 1000);
⚠️ Important Note:

Make sure to allow multiple downloads and pop-ups for takeout.google.com in your Chrome settings.
